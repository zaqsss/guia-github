Introdução ao GitHub


O que é GitHub e qual é a sua principal funcionalidade?

O GitHub é uma plataforma baseada na web que utiliza o Git para controle de versão. Ele permite que desenvolvedores hospedem, revisem e colaborem em projetos de software, facilitando a gestão de código fonte e a colaboração entre equipes de desenvolvimento.


Quais são as diferenças entre Git e GitHub?

Git é um sistema de controle de versão distribuído usado localmente para gerenciar históricos de código, permitindo operações como branches e merges. GitHub, por outro lado, é uma plataforma web que hospeda repositórios Git na nuvem, facilitando a colaboração através de pull requests, issues e gestão de projetos.
