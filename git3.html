Repositórios


O que é um repositório no GitHub?

Um repositório no GitHub é um local onde todos os arquivos do seu projeto são armazenados. Ele também registra o histórico de todas as alterações feitas nesses arquivos, permitindo que você e outros colaboradores trabalhem juntos no projeto.


Como criar um novo repositório no GitHub?

Para criar um novo repositório no GitHub:
  1. Entre na sua conta do GitHub.
  2. Clique no ícone "mais" (+) no canto superior direito e selecione "New repository".
  3. Escolha um nome para o repositório, defina se será público ou privado, e adicione um arquivo README se desejar.
  4. Clique em "Create repository".


Qual é a diferença entre um repositório público e um repositório privado?

Um repositório público é acessível por qualquer pessoa na internet, permitindo que outros vejam, baixem ou contribuam com o código. Já um repositório privado só pode ser acessado por você e pelas pessoas que você autorizar, garantindo mais controle sobre quem vê e interage com o projeto.
